# redirects

Данный плагин позволяет создавать 301 редиректы для нужных адресов, это пригодится при изменении структуры сайта.
-------------------

Возможности плагина:
- Возможность создавать редиректы от старых ссылок к новым
- Ведется список созданных ссылок
